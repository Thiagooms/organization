# Organization Frontend

React + Tailwind CSS frontend para a plataforma de organização acadêmica.

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

Acesse `http://localhost:3000`

## Build

```bash
npm run build
```

## Tecnologias

- React 18
- Vite
- Tailwind CSS
- React Router v6
- Axios
- Lucide Icons
